# 🎮 LearnOpenGL 日本語ドリル技術書

> **learnopengl.com** の完全日本語版ドリル教材。読んで・解いて・手を動かして OpenGL を習得しよう！

---

## 🎯 この教材の使い方

```
1. 📖 解説を読む        → 概念・理論の理解
2. 💡 ポイント確認      → 重要事項の整理
3. ✏️ ドリルを解く      → 理解度チェック
4. 🔨 実践課題に挑戦    → 手を動かして体得
5. ✅ 解答で答え合わせ  → 誤りを修正
```

## 🛠️ 必要な環境

| ツール | 用途 |
|--------|------|
| C++ コンパイラ（GCC / MSVC / Clang） | ソースのビルド |
| CMake | ビルドシステム |
| GLFW | ウィンドウ・入力管理 |
| GLAD | OpenGL 関数ローダー |
| GLM | 数学ライブラリ |
| stb_image | 画像読み込み |
| Assimp | 3D モデル読み込み |

## 📚 章構成

| 章 | テーマ | 内容 |
|----|--------|------|
| 🟢 1章 | はじめに | OpenGL の概要と学習の準備 |
| 🔵 2章 | 入門編 | ウィンドウ作成〜カメラ操作まで |
| 🟡 3章 | ライティング編 | Phong 照明モデル・マテリアル・光源 |
| 🟠 4章 | モデル読み込み編 | Assimp・メッシュ・Model クラス |
| 🔴 5章 | 高度な OpenGL 編 | テスト・ブレンド・FBO・インスタンシング |
